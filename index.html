Observaciones

Este espacio contiene texto largo para practicar la propiedad overflow. Cuando el contenido supera el alto máximo definido, el navegador puede mostrar una barra de desplazamiento para evitar que el diseño se rompa.

También podemos controlar el espacio interno con padding, el espacio externo con margin y los bordes con border. Esto permite que la página sea más clara, ordenada y fácil de leer para el usuario.
